# Security Policy

## 脆弱性の報告方法

このリポジトリで脆弱性を見つけた場合は、**公開 Issue に書かないでください**。
GitHub の private vulnerability reporting（非公開の脆弱性報告）を使って報告してください。

[https://github.com/tan3159/TradeScripts/security/advisories/new](https://github.com/tan3159/TradeScripts/security/advisories/new)

上記リンクから Security Advisory を作成すると、報告内容はリポジトリの管理者にのみ非公開で共有されます。

## 対応方針

- 報告を受領後、内容を確認し必要に応じて修正を行います
- 修正版のリリース後、Security Advisory を公開します
