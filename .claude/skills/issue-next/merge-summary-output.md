# マージ完了時の所要時間サマリ出力（#2313）

STEP 6 のマージ完了後・Issue やることチェックボックス更新の前に実行する。

```bash
tidd merge-summary report issue-<N> --pr <PR番号>
```

出力された Markdown 表（`## 所要時間サマリ`）をそのままメインセッションへの報告に含める。

バッチモード（複数 Issue 連続消化）の場合は、Issue ごとの `合計` 行の値を
`(issue番号, 合計時間)` として控えておき、キューが空になった最終報告の直前に以下を実行して
ロールアップ表も報告に含める:

```bash
tidd merge-summary rollup '[["#N1","5分00秒"], ["#N2","3分10秒"]]'
```
