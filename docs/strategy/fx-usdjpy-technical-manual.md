## 1. 使用インジケーターと環境認識

### 1.1 チャート構成

- **20EMA（青）**: 早期警報
- **50EMA（赤）**: 警報ライン（15分足）・押し目買い・戻り売りゾーン
- **200EMA（オレンジ）**: 長期トレンド・構造の決定線
- **SMC**: 供給（赤帯）/需要（緑帯）ゾーン
- **SQZMOM**: モメンタム方向の確認

### 1.2 MAの向き判定

- **上昇**: 現在値 > 8本前（15分足で2時間前）の数値
- **下降**: 現在値 < 8本前（15分足で2時間前）の数値
- **不一致**: 50MAと200MAの向きが異なる場合は、優位性が低いため静観を推奨

## 2. トレード戦略詳細

### 2.1 戦略1：15分足 MAプルバック

- **条件**: 価格が200MAの順張り側にあり、50/200MAが同方向
- **通常モード**: 50MA付近での反転（ヒゲ、包み足、ダブル・トリプル構造)でエントリーを検討
- **初動モード**: 200MA転換後の一発目の押し。ロット1/3。一度損切りとなった場合は、その日のセットアップは終了と判断
- **ブースト**: 1h足PO時、1h足の20MA-50MA間（15分足で見ると80MA-200MAの深い押し）で発生した場合は、利確目標を1h足の高安まで引き伸ばすことを検討

#### 2.1.1 初動モード（15分足 200EMA転換一発目）の定義とルール

1. **判定ロジック（客観的な数値基準）** — 長期的なトレンド転換の「一発目の押し・戻り」を機械的に判定するため、以下の数値を参照する
   - 上昇初動の判定条件（以下の両方を満たす）
     - ① 200EMA(現在) > 200EMA(8本前)
     - ② 200EMA(8本前) ≤ 200EMA(16本前)
   - 下降初動の判定条件（以下の両方を満たす）
     - ① 200EMA(現在) < 200EMA(8本前)
     - ② 200EMA(8本前) ≥ 200EMA(16本前)
2. **エントリー条件**
   - 上記判定後、最初に来る「50EMA」への接触のみを対象とする
   - エントリーのトリガー（ヒゲ、包み足、Wボトム/トップ等の実体確定）は通常モードと共通
3. **資金管理と損切り**
   - ロットサイズ: 通常時の 1/2〜1/3 に縮小
   - 損切り（SL）: 直近のスイング高値・安値の更新、または 200EMA をローソク足実体で明確に割り込んだ場合
   - 試行回数: 1つのトレンド転換につき原則1回限定。一度損切りとなったら、その日は同方向への初動狙いは終了（深追いを避ける）。ただし以下の例外あり
     - 例外を適用する大前提1: 1h環境認識がまだ崩れていない（PO継続＋直近高安未否定）
     - 例外を適用する大前提2: 直前の損切りが「想定レンジ内のノイズ」であること → 環境が変わっていたら再トライ禁止
     - 再トライ発動条件（以下すべて満たしたときのみ → 成行 or 指値で再エントリー）
       - ① 15mで直前エントリー方向に再度の高安更新
       - ② 15m 200EMAをタッチ or 明確反発
       - ③ 反発足が実体優勢（ヒゲより実体が大きい）
       - ④ 直前損切りから3本以上経過
     - 再トライ禁止条件（以下が出たら再トライ不可）
       - 1hで包み足逆行確定
       - 1h直近高安を否定
       - SQZMOMゼロライン割れ（方向逆）
       - 指標30分以内
     - 再トライのロット数: 当初のさらに半分（1/4〜1/6程度）
     - 再トライ回数: 一度だけ許可。再々トライは禁ずる
4. **利確（TP）戦略**
   - 15分足レベルの直近高安では決済せず、1時間足・4時間足基準の節目まで保有を検討する
   - 15分足でのトレンド終了（MAの逆転等）まで「スイング移行」を見据えて大きく伸ばす

#### 2.1.2 15分足トレンド延長のブースト（利幅拡大戦略）

1時間足のパーフェクトオーダー（PO）が確定している局面は、通常の15分足レベルの反転を超えた
「大きな波」になる可能性が高い。ブースト適用時は以下のルールに従い利益の最大化を図る。

1. **エントリー条件の拡張** — 深い押しの許容: 通常は50EMA付近を基準とするが、ブースト時は
   1時間足の20EMA〜50EMA の間（15分足換算で80MA〜200MAの深い押し）まで押しても、
   1時間足のトレンドが崩れていない限り、反転サイン（ヒゲ、包み足等）をもってエントリーを許可する
2. **ブースト発動条件** — 以下すべて満たすとき → 半分利確後、残りをブースト管理へ移行
   - 想定方向へ強いブレイク（直近高安の明確更新）
   - SQZMOM強拡大
   - ローソク実体が連続して伸びる（勢い確認）
3. **ブースト中の基本方針** — TP固定は置かない / 構造が崩れるまで保有 / 利を伸ばすことを優先
4. **ブースト中の出口ルール** — 【強制終了】以下のどれかで全決済:
   - 15mで直近押し安値（戻り高値）を明確に割る
   - 15mで20EMAを実体終値で2本連続割れ
   - 1hで包み足逆行確定
5. **警告（即終了しない注意シグナル）** — これ単体では利確しない。構造崩れが出たら即終了:
   - SQZMOM縮小 / BB±2σ内に一時回帰 / 上ヒゲ（下ヒゲ）連発
6. **トレールの置き方** — 直近の15m押し安値（戻り高値）に更新型トレール。含み益2R超えたら建値＋α固定も可
7. **指標発表時の継続ルール（例外処理）** — 通常、指標30分前は全決済とするが、ブースト中かつ
   以下の条件をすべて満たす場合のみ、SLを設定しての持ち越しを許可する
   - 継続条件: 含み益がリスク額（損切り幅）の2倍以上あること / SLを建値以上の優位なポイント（直近の安値/高値の裏など）に移動済みであること
   - リスク管理: 指標時のスプレッド拡大を考慮し、SLは通常より3〜5pips程度広めにバッファを持たせる
8. **利確目標の指針** — 第一目標: 1時間足レベルの直近高値・安値 / 第二目標: 「出口ルール」のいずれかが点灯するまで追随

### 2.2 戦略2：上位足の節目を狙った裏タッチ・ロールリバーサル

上位足（1時間足）で意識される強力な水平線（レジサポライン）やネックラインを、
下位足のMAがサポートするポイントで「順張り方向」に仕掛ける戦略。

#### 2.2.1 セットアップ条件（①・②を両方満たすこと）

- **① 1時間足の構造確定（環境認識）**
  - ダウ理論に基づくトレンド（高値・安値の更新）の発生
  - または、ダブルトップ（ボトム）、トリプルトップ（ボトム）のネックライン抜けによる構造の確定
- **② 節目とMAの一致（ゾーニング）**
  - 1時間足で抜けた「節目（水平線・ネックライン）」の裏側に価格が戻ってきた際、そのライン付近に
    **15分足または5分足の20EMA・50EMA・200EMAのいずれか（あるいは複数）** が重なっていること

#### 2.2.2 エントリー・トリガー（執行足: 15分足 / 5分足）

「節目 + MA」の重合ポイントで、以下のプライスアクションを確認して実体確定でエントリー:

- **下ヒゲ（上ヒゲ）** によるラインの反発確認
- **包み足** による方向感の否定と順張り方向への回帰
- 短期的な **ダブルボトム（トップ）・三尊（逆三尊）** 等のミニチャートパターンの形成

#### 2.2.3 成功のポイント

- 1時間足の「抜けた勢い」が強いほど、裏タッチ（ロールリバーサル）の信頼性は高まる
- SQZMOMがエントリー方向に強い色（勢い）を示しているか必ず確認すること

### 2.3 ハイブリッド戦略（先行＋本命）

強いブレイクアウト発生時に「機会損失」を防ぎつつ、期待値を最大化するための分割エントリー手法。

#### 2.3.1 先行エントリー（打診・お守り玉）

- **目的**: 戻らずに上昇する「最強のトレンド」への参加権確保
- **条件**: 1時間足の節目を15分足実体で明確にブレイクし、SQZMOMが強拡大していること
- **ロット**: **通常の 1/4〜1/5**（極小ロット）
- **損切り(SL)**:
  1. **ブレイク確定足の安値**（ロングの場合）から 2〜3pips 下（ショートは逆）
  2. **否定のサイン（包み足）**: 次以降の足がブレイク足を実体で包み込んだら無条件で撤退

#### 2.3.2 本命エントリー（ロールリバーサル）

- **条件**: 戦略2（リテスト）に基づき、本来の優位性があるポイントまで引き付けた際
- **ロット**: 先行玉を差し引いた「本来の予定ロット」
- **運用**: 本命が約定した時点で、先行玉のSLも本命のSL位置へ引き下げ（または建値へ引き上げ）てリスクを一元化する

### 2.4 利確と損切り（戦略1・戦略2共通）

#### 2.4.1 損切り

- **「逆指値（SL）」は、万が一の急変に備えて「ヒゲの少し外側」に** 物理的に置いておく
- **「手動撤退」は、SLに掛かる前であっても「ローソク足の実体が根拠（MAやライン）を割り込んで確定」** したらその場で行う

#### 2.4.2 利確

- **必須条件:** エントリー時に算出するリスクリワード（RR）が **1:1.5（1.5R）以上** 確保できることを前提とする
  - システムのゲート値も 1.5R（Gem プロンプト側の 2R 表記との矛盾はユーザー決定で 1.5R に統一。
    通知には 2R 確保可否を併記する。`docs/decisions/2026-07-22-fx-rr-gate-1.5r.md`）
- **TP1（第1目標）:** RR 1:2 の位置、または直近高値・安値の手前（どちらか近い方）
- **TP2（第2目標）:** ブースト時や初動モードにおいて、上位足（1h/4h）の節目まで
- **判断基準:** 到達前に勢いが弱まった場合（SQZMOMのドット色変化等）は、R倍数にこだわらず分割決済を優先する
- **利確の壁:** エントリー方向に **ボリンジャーバンド±2σ** が位置している場合、それを強力な抵抗帯とみなし、TP をその手前に設定する
- **逆張り警戒:** 価格が±2σを大きく超えてバンドウォークから外れた場合は、調整（戻り）のサインとして、利益が乗っていれば即座に利確を検討する

## 3. アラートの定義と運用ルール

チャート監視の効率化とメンタル維持のため、アラートを以下の役割で使い分ける。
**「アラート鳴動＝エントリー」ではなく、「アラート鳴動＝詳細分析の開始」** であることを徹底する。

### 3.1 ラインごとの役割分担

- **20EMA（早期警報ライン）:**
  - 役割: セットアップの可能性を示唆
  - アクション: PCまたはスマホでチャートを開き、現在の相場環境（200MAとの位置関係、各MAの傾き）を再確認する
- **50EMA（警報ライン / ストライクゾーン入口）:**
  - 役割: エントリー検討ゾーンへの到達
  - アクション: ローソク足の形状（ヒゲ、包み足、チャートパターン）の監視を開始する。SQZMOMやSMCのゾーンとの重なりをチェックし、シナリオの精度を高める

### 3.2 アラート鳴動後のチェックリスト（ルーティン）

アラートが鳴った際は、以下の項目を順に確認する:

1. **MAの向き**: 50MAと200MAが同方向を向いているか？（8本前比較）
2. **200MAの位置**: 価格が200MAの内側（順張り方向）にあるか？
3. **モメンタム**: SQZMOMの色や勢いはエントリー方向に一致しているか？
4. **指標確認**: 直近1時間以内に重要指標がないか？

### 3.3 運用上の注意点

- **無効なアラートの削除**: 週末や相場急変後など、環境認識が変わった場合は、古いアラートを一度リセットし、現在の構造に基づいた価格に再設定する
- **アラートによる「飛び乗り」禁止**: アラートはあくまで「監視の合図」であり、必ずローソク足の実体確定を待ってから判断を下す

## 4. 資金管理と防衛ルール（絶対厳守）

トレードの目的は「資産を守りながら、優位性を積み上げること」であり、以下の数値を機械的に適用する。

### 4.1 ロット計算基準

- **基準（アンカー）**: 損切り位置まで **10pips ＝ フルロット**（資金100万円に対し10枚/100,000通貨）とする
- **調整ルール:** 損切り幅に応じて、1トレードあたりのリスク額が一定になるよう調整する
  - 損切り幅 10pips: フルロット
  - 損切り幅 20pips: 1/2ロット
  - 損切り幅 40pips: 1/4ロット
- **先行玉の扱い**: ハイブリッド戦略の先行玉は、常にこの計算結果の **20%（1/5）** 程度に抑える
- **計画的分割エントリー（打診と本命）:** 強力な節目やゾーンにおいて、一度に全ロットを投入せず、以下の形でポジションを構築することは許可する
  - **打診（早期）エントリー:** 有力な節目に到達した際、反転の兆しを見て少なめのロット（例: 予定の1/4〜1/2）で入る。この場合も上記の調整ルールは絶対に守る
  - **本命（深い押し・戻り）エントリー:** 予定していた最も優位性の高いポイント（50EMAやSMCゾーンの深部など）まで価格が到達した際に、残りのロットを追加する
  - **条件:** 分割エントリーを行う場合も、全ポジション合計のリスク（損切り時の損失額）が、当初の予定リスク許容範囲内に収まっていることを絶対条件とする
- **ピラミッティング（増し玉）:** 利益が乗り、次の押し目・戻り目（MAタッチ等）で根拠が重なった場合のみ、ストップを建値に移動した上でさらに利益を積み増すための追加は推奨する（フルロットではない場合や部分利確後の再チャンス）
- **ナンピン厳禁:** 当初のシナリオにない場所で、含み損を解消する目的での場当たり的な追加（ナンピン）は厳禁とする

### 4.2 連敗時のロット制限（防衛）

感情的なトレードや相場との不一致を防ぐため、以下の制限を設ける:

- **2連敗時:** 翌日のトレードロットを 1/2 に縮小する
- **3連敗時:** 一旦トレードを完全に停止し、マニュアルとの乖離がないか環境認識を再点検する

### 4.3 負けないための技術（撤退とトレール）

- **建値ストップ:** 含み益が 15pips に達した時点で、逆指値を建値（または微益）に移動し、そのトレードでの損失リスクをゼロにする
- **微益撤退の優先:** 勢いが弱まった、あるいは指標が迫っている場合は、目標値に届かずとも微益で撤退し、資金を回転させることを優先する
- **トレール注文:** 15分足のトレンドが継続する場合、15pips幅程度のトレールを活用し、利益の最大化を狙う
- **指標30分前ルール:** 重要指標の30分前には、損益に関わらず、全ポジションの決済（全利確・全損切り）を強く推奨する。ただし以下の例外がある
  - **指標発表時の例外ルール:** 含み益が十分（目安としてリスク額の2倍以上）乗っており、SLを建値以上の優位なポイントに移動済みの場合は、指標30分前であってもポジションの継続を許可する。ただし、指標直前のスプレッド拡大を見越し、SLは通常より数pips余裕を持たせること
- **指標直後の静観:** 重要指標通過後、スプレッドが安定し、かつ指標発表から最低15分（15分足実体1本分）経過して方向性が確定するまで、新規エントリーを禁止する

## 5. エントリー見送りの基準（触らない条件）

以下の状況ではリスクがリワードを上回るため、見送りを強く提案する:

- 200MAの外側（逆張り）でのエントリー
- MAの向きが不一致、または横ばい（方向感欠如）
- 形状の乱れ（長い上下ヒゲ、実体が極端に小さい足の連続）
- 重要指標の直前、および週またぎ
- ボリンジャーバンド（±2σ）の活用: バンドがスクイーズ（収束）している時はレンジと判断し、戦略1を見送る
- 週足レベルのEMAとの衝突（EMAがチャート上は線に見えても実態は太い帯であるポイント）
- 上位足トレンドの終焉が懸念される節目
